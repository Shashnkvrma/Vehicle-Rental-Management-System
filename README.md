# Vehicle Rental Management System

A web-based Vehicle Rental Management System for booking and managing rental vehicles.

## Description
This is a college project built using HTML, CSS, and JavaScript.  
The system provides a user interface for customers to view vehicles and book them, while admins can manage vehicle listings.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User Login Page
- User Registration Page
- Vehicle Listing Page
- Responsive UI Design

## Project Status
Currently this project contains only the frontend interface. Backend functionality will be added later.
